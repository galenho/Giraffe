-------------------------------------------------------------
		Giraffe简介
-------------------------------------------------------------
    在crossover框架基础上开发的一个分布式MMORPG游戏服务器项目

	开发者可以利用现成代码模板，快速搭建类似bigworld引擎的多进程的MMORPG集群架构，也可以轻松地搭建其他游戏类型的集群架构。

-------------------------------------------------------------
		使用方法
-------------------------------------------------------------
一、windows 10下的使用

	1.	安装mongodb 4.0
	2.	运行run.bat
	注：初次运行需要在db_server的控制台下输入命令insert account来新增10000个帐号
	3.	运行run_client.bat

二、centos7.6下的使用

	1.	安装mongodb 4.0
	2.	在crossover目录下执行chmod -R * 750
	3.	运行./run.sh
	注：初次运行需要在db_server的控制台下输入命令insert account来新增10000个帐号
	4.	运行./run_client.sh

三、源码的打开

	推荐用luastudio, 直接打开crossover.luaprj工程文件即可
